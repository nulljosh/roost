# roost -- Agent Notes

BC real estate listings PWA. Renamed from zillow-clone. React + Vite, map + list views.

## Overview
Real estate listing browser for British Columbia. Pulls MLS data and displays on interactive map with list/grid views.

## Stack
- React + Vite
- Leaflet/Mapbox for maps
- Dark Editorial design system (BC gov blue variant)

## Quick Commands
- npm run dev -- start dev server
- npm run build -- production build

# roost

![version](https://img.shields.io/badge/version-v1.0.0-blue)

BC real estate listings PWA with interactive maps, photo galleries, and filtering.

## Features

- Leaflet map with Zillow-style price pill markers
- Photo carousels with swipe and lightbox gallery
- Search, filter (beds/baths/type/city/price), sort
- Favorites (localStorage-backed)
- Similar homes recommendations
- 50 mock BC listings with enriched data

## Run

```bash
npm install && npm run dev
npm run build
```
Vercel or `npm run build` for static hosting.
